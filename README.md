### Motivating the dot product
 
$\bar{z} w = (a - bi)(c + di) $
  
  $= (ac + bd) + (ad - bc)i $
  
$= |z|e^{-\alpha i}|w|e^{\beta i} $
  
$= |z||w| \cos \theta + (|z||w| \sin \theta)i $
  
  So $z \cdot w = \frac{1}{2} \left( \bar{z}w + z \bar{w} \right)$
